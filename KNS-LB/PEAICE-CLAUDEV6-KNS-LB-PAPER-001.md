## 2. Two-layer decomposition: full proofs

### 2.1 Lemma 3.1 (overlap is a statistic) — full proof. `FORMAL` · discharges **OB-KNS-3**

**Statement.** Let 𝒟 = (T, Y) be an incidence datum (finite δ-tube family with
measurable shadings, up to ambient isometry) and let

```text
S(𝒟) := ( μ(T,Y), Δ_max(T), {μ(T,Y)(x)}_{x∈U}, |U(T,Y)| )
```

be its overlap statistic (all entries defined purely from set data, [GWZ26]
eqs. (3),(7)). Let a *placement datum* be P = (H, A, ψ): separable Hilbert
space, closed register subspace A with orthogonal projection π_A, unit state
ψ, and leakage ℓ_off(ψ) = ‖(I−π_A)ψ‖². Then **no function F satisfies
F(S(𝒟)) = ℓ_off across admissible encodings**: for every 𝒟 and every
λ ∈ [0,1] there is an admissible placement with ℓ_off = λ.

*Proof.* (i) *Statistic measurability.* Each entry of S(𝒟) is a functional of
the sets {(T, Y(T))} alone; the map 𝒟 ↦ S(𝒟) factors through the σ-algebra
generated by incidence data. No Hilbert-space object appears. (ii) *Placement
freedom, state side.* Fix any encoding-eligible H and register A with
dim A ≥ 1, dim A^⊥ ≥ 1. Pick unit u ∈ A, v ∈ A^⊥ and set
ψ_λ = √(1−λ) u + √λ v: the incidence datum is untouched, ℓ_off(ψ_λ) = λ.
(iii) *Placement freedom, register side.* Dually, fix any canonical encoding
ψ(𝒟) (e.g. the l2c_probe center mode); for any λ choose a register A_λ whose
projection satisfies ‖π_{A_λ}ψ‖² = 1−λ (rotate A). Either freedom alone kills
any factorization S(𝒟) → ℓ_off. ∎

μ and Δ_max therefore *underdetermine* π_A and Action(C): they are observable
from union footprints; placement is a declaration on the lawful sector.

## 4. CC-BL-001a: split verdict — scoped theorem + explicit countermodel. Discharges **OB-KNS-1**

Original claim (scaffold §5): *misalign center C under π_A; ℓ_off increases
monotonically with offset.* Result of this pass: **REFUTED as stated,

**Theorem 4.1 (monotone leakage, unimodal class).** `FORMAL`
Let φ ∈ L²(ℝ^d) be radial, log-concave, ‖φ‖=1. Register A = span{τ_{C₀}φ}
(rank-one, "one detector head"); misplaced state ψ_δ = τ_{C₀+δe}φ. Then

```text
ℓ_off(δ) = 1 − ρ(δ)²,   ρ(δ) = (φ ⋆ φ̌)(δe),
```

and ρ is radially nonincreasing, hence ℓ_off is monotone nondecreasing in
δ ≥ 0 — strictly increasing when φ is strictly log-concave (Gaussian:
ℓ_off(δ) = 1 − e^{−δ²/(2w²)}).

*Proof.* φ ⋆ φ̌ is log-concave (convolution of log-concave densities is
log-concave, Prékopa 1973 / Prékopa–Leindler) and radial; a radial log-concave
function attains its maximum at 0 and is nonincreasing along rays. Strictness
for the Gaussian is the closed form. ∎

**Countermodel 4.2 (multi-well register).** `FORMAL + NUMERICS`

Probe receipt (w=0.35, L=2.0): peak ℓ = **0.9662 at δ = 1.0**, returning to
≈0 at δ = 2.0; `D5_twomode_monotone = false`. Physical reading: multi-detector

**L3 requires (arithmetic side, OPEN · FORCED post-V6.5):** prime-carrying
data — lengths log p, weights Λ(p^k)p^{−k/2} — entering a self-adjoint
pair whose determinant/counting reaches Riemann–von Mangoldt density

walls L2-5, WP5-OBS-1, HS-CORRIDOR, WP5-OBS-2 — all Weyl-stability faces of
the n⁴ ladder).

**Krein face (new wall entry, `PROPOSED`: KREIN-RANK1).** For rank-one moves
A → A + λ(·,φ)φ the Krein spectral shift is exactly the *phase shift of the
spectral-measure pair* (μ,ν): 1 + πλ𝒦μ = exp[𝒦u] = [1 − πλ𝒦ν]^{−1}
([P96] §3, Lemma 3.1). Poltoratski Theorem 4.1: all self-adjoint rank-one
perturbations of A are pure point ⟺ σ(A) is countable; and the essential
spectrum is Weyl–von Neumann stable under such moves ([P96], proof of 2⇒1;
Corollary 4.4 extends to trace class). Consequence: on the K_σ/n⁴-ladder
side, no bounded, trace-class, or KSSF-mediated rank-one injection changes
the counting class — the KSSF face of the same wall family. Therefore prime
content must live in the **measure-pair / function-system data itself**: the
Nyman–Beurling / Báez-Duarte program (completeness of {ρ(θ/n)} span in
L²(0,1)) is the preferred carrier, where "placement" is a completeness
statement, not a perturbation of a fixed geometric operator. KNS's Π_½
register is the declared home for that completeness ledger; **supplying the
pair (μ,ν) with prime phase shift u is precisely what remains OPEN.**
